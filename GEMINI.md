# RumCoCo Project Constitution

## Project Identity
**RumCoCo** (Compliance, Finance, & Operations OS)
A business operating system for Florida/Caribbean markets.

## Design Language
**"Tropical Professional"** (Miami-Vice meets serious finance)
- **Colors:**
  - Neon Pink: `#FF6EC7`
  - Charcoal: `#262626`
  - Palm Green: `#4C9A2A`
- **Font:** Roboto

## Tech Stack
- **Framework:** Next.js (App Router)
- **Language:** TypeScript (Strict)
- **Styling:** Tailwind CSS
- **Database:** Prisma (PostgreSQL)
- **Icons:** Lucide React

## Coding Rules
1. **Always use TypeScript interfaces.**
2. **Use Server Actions for backend logic** (no separate Express server).
3. **"Mobile-First" responsive design** using Tailwind utility classes.
4. **Favor "composition" over inheritance.**
